A fully custom, hand-coded portfolio built with zero frameworks — just HTML, CSS, and vanilla JavaScript. Designed to stand out on LinkedIn and leave a lasting impression on recruiters and hiring managers.

---

## ✨ Features

- Custom animated cursor with violet/amber hover states
- Scroll-triggered reveal animations on every section
- Animated stat counters that fire on scroll into view
- Filterable project grid by category (NLP, ML, Time Series, Deep Learning, and more)
- Responsive layout for mobile and desktop
- Subtle grid background, floating glow blobs, and noise texture overlay
- Zero dependencies — no React, no Tailwind, no build step

## 🚀 Deploy

This repo deploys automatically via **GitHub Pages** from the `main` branch.

To update the site, edit `index.html` and push to `main` — changes go live in under a minute.

```bash
git add index.html
git commit -m "update portfolio"
git push origin main
```
